An Intuitive Explanation of

Bayes' Theorem

$$P\left( A \right) =\frac { \left| A \right| }{ \left| U \right| }$$

$$P\left( B \right) =\frac { \left| B \right| }{ \left| U \right| }$$

$$P\left( AB \right) =\frac { \left| AB \right| }{ \left| U \right| }$$

$$P\left( { A }|{ B } \right) =\frac { \left| AB \right| }{ \left| B \right| } =\frac { \frac { \left| AB \right| }{ \left| U \right| } }{ \frac { \left| B \right| }{ \left| U \right| } } =\frac { P\left( AB \right) }{ P\left( B \right) }$$ $$P\left( { B }|{ A } \right) =\frac { \left| AB \right| }{ \left| A \right| } =\frac { \frac { \left| AB \right| }{ \left| U \right| } }{ \frac { \left| A \right| }{ \left| U \right| } } =\frac { P\left( AB \right) }{ P\left( A \right) }$$

$$ P\left( { B }|{ A } \right) =\frac { P\left( AB \right) }{ P\left( A \right) } \quad \Rightarrow \quad P\left( AB \right) =P\left( { B }|{ A } \right) \cdot P\left( A \right) $$ $$ P\left( { A }|{ B } \right) =\frac { P\left( AB \right) }{ P\left( B \right) } \quad \Rightarrow \quad P\left( AB \right) =P\left( { A }|{ B } \right) \cdot P\left( B \right) $$ $$ P\left( { A }|{ B } \right) \cdot P\left( B \right) =P\left( { B }|{ A } \right) \cdot P\left( A \right) $$ $$ P\left( { A }|{ B } \right) =\frac { P\left( { B }|{ A } \right) \cdot P\left( A \right) }{ P\left( B \right) } $$
